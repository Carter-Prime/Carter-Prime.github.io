GamerVibe

GamerVibe is a Instagram like platform for sharing screenshots of in game images and gamer's personas.

The backend was built using Node.js and SQL as the database. Frontend was HTML/CSS and JavaScript without the use of frameworks or libraries.

The aim of this project was to understand and practice the fundamentals of building a web application.

LingoKnight

LingoKnights is an Android application for children that helps then learn new languages in a fun environment.

The user is shown pictures and the name of the image in the language of choice. The user is then quizzed at the end with a score given.

The focus of the project was to learn Kotlin and applying Android SDK fundamentals.
